# lofi-stream-odysee

24/7 lofi stream to Odysee with an underwater/deep sea theme.

## Live Stream

Watch at: [odysee.com/@lofistream](https://odysee.com/@lofistream) (update with actual channel)

## Theme: Deep Sea

A submarine porthole view into the deep ocean, with bioluminescent creatures, fish, coral, and ambient underwater audio.

## Quick Start

```bash
# View locally
cd docs && python3 -m http.server 8080
# Open http://localhost:8080
```
